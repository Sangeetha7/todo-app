# 📝 To-Do List App  

A simple command-line **To-Do List app** written in Python.  
It allows users to add, remove, and view tasks with data stored in a JSON file.  

---

## 🚀 Features  
- Add tasks  
- Remove tasks  
- Save tasks between sessions
